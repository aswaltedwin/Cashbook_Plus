# Cashbook+  

Cashbook+ is a lightweight, web-based personal finance tracker that allows users to manage their cash inflows and outflows efficiently. Designed to work on both desktop and mobile devices, it provides an intuitive interface to log transactions, visualize balances, and export data in PDF or Excel formats.  

## Features  

- **Multiple Cashbooks:** Maintain separate cashbooks for different purposes (personal, school, work, etc.).  
- **Transaction Management:** Add, edit, and delete entries for cash in and cash out.  
- **Summary & Insights:** View total cash in, total cash out, and current balance at a glance.  
- **Filters:** Search by date or keyword to quickly find specific transactions.  
- **Export Options:** Export transactions to **PDF** (formatted for clarity) or **Excel**.  
- **Theme Support:** Switch between light and dark modes.  
- **Mobile-Friendly:** Fully responsive interface to use on smartphones.  

## Technology Stack  

- **Frontend:** HTML, CSS, JavaScript  
- **Data Storage:** LocalStorage (with optional API integration for server-side storage)  
- **PDF Generation:** jsPDF  
- **Excel Export:** SheetJS (xlsx)  
- **Hosting:** Can be deployed on free hosting services like Vercel or Netlify  

## Usage  

1. Open the app in your browser.  
2. Create or select a cashbook.  
3. Add transactions using the entry form.  
4. Use filters to search specific transactions.  
5. Export your cashbook to PDF or Excel for offline records.  

## Deployment  

This project can be easily deployed on free hosting platforms such as **Vercel** or **Netlify**. Simply push your repository to GitHub and link it with the hosting platform.
